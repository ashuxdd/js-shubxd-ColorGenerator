# Color Generator

A random color generator that produces hex colors and applies them to the page background.  
Users can generate colors, copy the hex value, or manually enter their own hex code.  
This project demonstrates random number generation, DOM updates, and user interaction.

## Features
- Generate random hex colors
- Display and apply the color visually
- Copy hex code to clipboard
- Accept manual hex input
- Smooth UI and color transitions

## Technologies Used
- HTML5
- CSS3
- JavaScript (ES6)

## How to Run
1. Clone or download the repository.
2. Open `index.html` in your browser.
3. Click **Generate** to create random colors.

## Project Structure
- `index.html` – base UI layout
- `style.css` – styles for the generator
- `script.js` – logic for generating and applying colors
